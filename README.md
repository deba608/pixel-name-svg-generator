# Pixel Name SVG Generator

## Requirements
- Python 3.x

## Generate
python generate_pixel_svg.py --name "Debashish Pradhan" --subtitle "AI Engineer & Developer" --out name-animation.svg

## Windows shortcut
Double-click make_svg.bat

## How to use
1. Download the ZIP or clone this repo.
2. Double-click the batch file for the easy shortcut.

Easy shortcut: 
double-click `make_svg.bat`.

It will prompt you for:
- Name
- Subtitle

Then it creates an animated SVG only.
